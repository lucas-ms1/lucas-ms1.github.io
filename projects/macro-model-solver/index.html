What this tool does

Macro Model Solver provides an interactive interface for solving and visualizing canonical macroeconomic models. The live app lets you switch between multiple models and inspect implied dynamics and steady-state relationships. This static page mirrors that structure with three client-side tabs (one per model), using the app’s default parameterization at export time.

  • Model 1 — Consumption-Savings: CRRA utility with an exogenous income shock.
  • Model 2 — Robinson Crusoe: capital accumulation with a TFP shock.
  • Model 3 — Labor Supply: labor/leisure choice with a wage shock.

Methodology (what “solving” means)

For each model, the app takes a parameterization (in this demo: the app’s default parameters at export time), computes the relevant equilibrium/solution objects (via value function iteration in the interactive app), and then visualizes outcomes such as policy functions and simulated paths.

The figures below are not screenshots; they are exported Plotly objects, so you can hover and zoom to inspect values. Any differences between this demo and the live app typically come from changed defaults, updated packages, or upstream data revisions.

Full app features (interactive version)

  • Model switching across all three VFI-based macro models from one interface.
  • Editable structural parameters (for example, beta, sigma, transition probabilities, and grid size).
  • An advanced mode with extra controls and diagnostics that are not always shown in the simplified view.
  • A built-in risk-aversion estimator workflow for backing out an implied CRRA coefficient from user inputs.
  • Reruns and recomputation after parameter changes, including updated plots/tables.

What this static page does not include

  • No live solve button: all outputs here are precomputed snapshots.
  • No interactive estimator flow: the risk-aversion estimation UI is described, not executable, on this page.
  • No advanced-mode toggling: only the exported state is shown.
  • No file upload or custom-run persistence from the browser.

How to run the full interactive version

Clone the repo and run the Streamlit app locally:

git clone https://github.com/lucas-ms1/ECO-317-Project-2---Model-Solver.git
cd ECO-317-Project-2---Model-Solver

python -m venv .venv
# Windows:
.venv\\Scripts\\Activate.ps1
# macOS/Linux:
# source .venv/bin/activate

pip install -r requirements.txt
streamlit run app.py
