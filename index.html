Projects & Research

Selected research and technical work.

Butler Co. Parks — Economic Impact Report

Spatial econometric analysis quantifying the economic and health impacts of public parks. Integrates GIS, time-series, and health data to estimate property value capitalization.

GitHub repo

Bargaining Unit SOC Classification Project

Automated NLP pipeline to classify NLRB bargaining unit descriptions into Standard Occupational Classification (SOC) groups. Integrates Hybrid Retrieval (BM25 + Sentence Transformers) and LLM-based multi-label classification. Uses Prediction-Powered Inference (PPI) to correct model bias and estimate temporal prevalence trends.

GitHub Repo

DAAHO Metadata Pipeline

AI-powered extraction pipeline for historical archives. Uses GPT-4o for OCR text processing and LOC15-compliant metadata extraction with controlled vocabulary validation.

GitHub repo

Double Machine Learning for Dynamic Panel Data

Developed a DML framework for inference in partially linear dynamic panels with high-dimensional controls. Proposed a blocked-time cross-fitting scheme with time buffers to respect serial dependence, combined with Neyman-orthogonal scores to eliminate regularization bias from ML nuisance estimators (e.g., Lasso, Random Forests).

Econometrics DML Python

Intermediary Constraints & Asymmetric Sentiment Transmission

Investigated how intermediary balance-sheet health acts as an "on/off switch" for sentiment-driven price crashes. Estimated an endogenous two-state Markov Regime Switching (MRS) model to identify constrained periods, documenting that negative household sentiment triggers sharp reversals in high-inventory-risk stocks (High DTC) specifically when liquidity providers are capital-constrained.

Market Microstructure Asset Pricing STATA/R

Mean Field Game Modeling of Overconfidence

Simulation framework combining Mean Field Games with filtering (Kalman/EKF) and heterogeneous agents to study endogenous overconfidence and sentiment cascades across market regimes.
