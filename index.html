About

I’m a senior at Miami University studying Political Science, Philosophy, and Economics, focused on rigorous mathematical modeling in finance and economics. My current work uses Mean Field Games and filtering (e.g., Kalman/EKF) to study belief dynamics, mispricing, and sentiment feedback in equity markets, alongside empirical analysis using U.S. market data. I’m looking for research roles and graduate programs in quantitative economics/finance, stochastic control, and applied math.

Projects

Selected work (more details and writeups coming soon).

Butler Co. Parks — Economic Impact Report

Spatial econometric analysis quantifying the economic and health impacts of public parks using multiple datasets and methods. Integrates GIS, time-series, health, and economic data to estimate property value capitalization and health outcome associations.

GitHub repo

DAAHO Metadata Pipeline

AI-powered metadata extraction pipeline for the DAAHO digital archive. Extracts OCR text and LOC15-compliant metadata from historical documents using GPT-4o, with controlled vocabulary validation, an interactive review interface, and CSV export.

GitHub repo

Buying the Farm: Conservation Easements and Property Values

A spatial econometric analysis (Spatial Durbin Model) estimating the capitalization effects of agricultural conservation easements on residential property values in Butler County, Ohio. Results indicate a significant positive premium for properties within close proximity to preserved farmland.

GitHub (repo link TBD)

Does "Woke" Go Broke? A Nike Event Study

Financial event study analyzing the impact of the Nike/Dylan Mulvaney partnership on shareholder value. Utilizes CAPM and Fama-French 3-factor models to test for abnormal returns, finding statistically significant negative Cumulative Average Abnormal Returns (CAAR) in the immediate event window.

GitHub (repo link TBD)

Simulating Social Learning in Networks

Agent-based simulation investigating how network topology (Erdos-Renyi vs. Barabasi-Albert) influences consensus formation. Compares DeGroot and Bayesian learning models to quantify the trade-off between information aggregation speed and accuracy in complex social graphs.

GitHub (repo link TBD)

Mean Field Game Modeling of Overconfidence in Financial Markets

A simulation framework combining Mean Field Games with filtering and heterogeneous agents to study endogenous overconfidence, sentiment cascades, and mispricing across market regimes.

GitHub (repo link TBD)

Academic Skepticism, Imitation Mechanisms, and Cultural Homogenization
